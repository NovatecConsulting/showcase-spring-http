= HTTP Client/Server Examples with Spring Boot

.Build Image (see https://docs.spring.io/spring-boot/docs/2.4.5/gradle-plugin/reference/htmlsingle/#build-image)
[source,bash]
----
./gradlew bootBuildImage
----

.Run
[source,bash]
----
docker-compose up
----
